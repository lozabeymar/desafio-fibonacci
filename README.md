# 🌿 Fibonacci en la Naturaleza: El Lenguaje de la Creación

Este proyecto web explora la **Sucesión de Fibonacci** y su fascinante presencia en el mundo natural, desde la filotaxis de las plantas hasta la geometría de las galaxias. 
Desarrollado como una herramienta interactiva para visualizar cómo las matemáticas dictan la eficiencia biológica.

## 🚀 Características
- **Explicación Teórica:** Análisis de la relación entre la serie de Fibonacci y el diseño natural.
- **Calculadora Interactiva:** Generador de niveles de la secuencia utilizando JavaScript.
- **Diseño Responsivo:** Interfaz optimizada para dispositivos móviles y escritorio con una paleta de colores claros y orgánicos.
- **Fundamentación Filosófica:** Reflexión sobre la convergencia entre la ciencia y el diseño divino.

## 🛠️ Tecnologías Utilizadas
- **HTML5:** Estructura semántica.
- **CSS3:** Diseño personalizado con Flexbox, Media Queries y variables modernas.
- **JavaScript (Vanilla):** Lógica de programación para el cálculo de la serie y manipulación del DOM.
